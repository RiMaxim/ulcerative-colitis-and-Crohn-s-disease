**Термины**

1) Inflammatory Bowel Disease (IBD) - Воспалительные заболевания кишечника (ВЗК). Это собирательный термин, который объединяет два хронических состояния, характеризующихся рецидивирующим (возвращающимся) воспалением желудочно-кишечного тракта. Ключевая особенность — воспаление вызвано аномальным ответом иммунной системы на бактерии кишечника (а не инфекцией). ВЗК включает болезнь Крона и язвенный колит. Важно не путать с СРК (синдромом раздраженного кишечника): при ВЗК есть структурные повреждения тканей и воспаление, а при СРК — только функциональные нарушения.

2) Ulcerative Colitis (UC) - Язвенный колит (ЯК). Хроническое воспалительное заболевание, поражающее только толстую кишку (ободочную и прямую). Воспаление начинается в прямой кишке и непрерывно распространяется вверх по толстой кишке, поражая только слизистую оболочку (самый внутренний слой стенки кишки). Ключевые характеристики: Поражается только толстая кишка (тонкая кишка здорова). Воспаление непрерывное (нет «островков» здоровой ткани). Частые симптомы: кровавая диарея, боль внизу живота, тенезмы (ложные позывы).

3) Crohn’s Disease (CD) - Болезнь Крона (БК). Хроническое воспалительное заболевание, которое может поражать любой отдел ЖКТ — от рта до ануса, но чаще всего терминальный отдел подвздошной кишки (тонкая кишка перед толстой) и толстую кишку. Воспаление затрагивает все слои стенки кишки (трансмуральное). Ключевые характеристики: Может быть в любом месте ЖКТ. Поражение сегментарное («прыгающее») — участки воспаления чередуются со здоровой тканью. Глубокие язвы, свищи (патологические ходы между органами), стриктуры (сужения).
